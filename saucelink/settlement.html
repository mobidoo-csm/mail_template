소스링크 정산 데이터 전달 요청

Web SDK / MMP 연동 고객사의 정산 처리를 위해
수기 정산 데이터 전달이 필요합니다.

{회사명} {담당자명}님, 안녕하세요.
모비두 고객성공팀 솔루션파트입니다.

앞서 안내드린 바와 같이, 소스링크 Web SDK / MMP 연동 방식의 경우
정산 데이터 반영을 위해 SFTP 기반의 정산 연동이 필요합니다.

현재 SFTP 기반 정산 연동이 완료되지 않은 상태로 확인되어,
도래한 정산 건에 대해 우선 수기 방식으로 정산 데이터 전달을 요청드립니다.

아래 필수 컬럼 및 작성 기준을 확인하신 후,
정산 데이터를 작성하여 회신 부탁드립니다.

📌 현재 정산 연동 상태 안내

Web SDK / MMP 연동은 성과 추적을 위한 연동 방식입니다.
다만 실제 주문 확정, 취소, 부분 취소 등 정산 기준 데이터는
고객사 내부 주문 시스템 기준으로 확인이 필요합니다.

현재 SFTP 기반 정산 연동이 완료되지 않아,
이번 정산 건은 정산 데이터를 수기 파일로 작성하여 전달 부탁드립니다.

1

🧾 필수 전달 컬럼

아래 항목을 포함하여 정산 데이터를 작성해 주세요.

컬럼명 설명
order_datetime 주문/취소 일시 (UTC 기준)
order_id 주문 ID
product_id 상품 ID
product_order_amount 주문/취소 금액
product_order_quantity 주문/취소 수량
order_confirm 정산 대상 여부 (Y / N)

※ 파일은 UTF-8 형식으로 작성 부탁드립니다.
※ order_confirm 값은 반드시 Y 또는 N으로 입력해 주세요.
※ 주문/취소 일시는 UTC 기준으로 전달 부탁드립니다.

︾
2

✅ 주문 상태별 작성 기준

주문 상태에 따라 아래 기준으로 작성 부탁드립니다.

정상 확정
Y		 전체 취소
N		 부분 취소
Y/N 행 분리

※ 부분 취소 건은 확정분(Y)과 취소분(N)을 각각 별도 행으로 분리해 주세요.
※ 확정분과 취소분의 금액 및 수량 합계는 원본 주문과 일치해야 합니다.
※ 취소 금액 및 수량은 음수(-)가 아닌 양수로 입력해 주세요.

📚 소스링크 정산 연동 가이드 보러가기

🔗 SFTP 자동 정산 연동이 필요하신가요?

추후 수기 전달이 아닌 자동 정산 처리를 원하시는 경우,
고객성공팀 솔루션파트로 문의해 주세요.

정산 연동 가이드 안내와 SFTP 연동 진행을 지원해 드리겠습니다.

또한 더 나은 서비스 제공을 위해 정산 연동 방식은
지속적으로 고도화할 예정입니다.
관련하여 의견이 있으시다면 언제든 편하게 말씀해 주세요.
