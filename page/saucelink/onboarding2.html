📖소스링크 주요 FAQ 📖

Q
캠페인과 링크의 차이가 궁금합니다.
캠페인은 링크를 생성·관리하는 단위이며,
링크는 상품과 크리에이터가 1:1로 매칭된 추적 URL입니다.
Q
분석 메뉴가 업데이트되지 않아요.
분석 데이터는 매시 20분에 업데이트됩니다.
예)  10:00~11:00 데이터 → 11:20 반영
11:00~12:00 데이터 → 12:20 반영
Q
크리에이터 초대 후, 크리에이터 페이지는 어떻게 확인하나요?
크리에이터
 관리자가 발송한 초대 이메일을 통해 가입을 진행한 후, 해당 페이지에서 캠페인 및 링크를 확인할 수 있습니다.
관리자
 관리자 계정으로는 페이지 확인이 불가하므로, 확인 필요 시 관리자용 크리에이터 계정을 생성하여 로그인해 주세요.
